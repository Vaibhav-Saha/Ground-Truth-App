# Ground-Truth-App

Install npm, node.
Install libraries using command 

npm install <library_name>

Such Libraries are:-

Mongoose,
Path, and
Express.

After Successful Installation of all the libraries 

Run

node server.js

And visit the locally hosted site-

http://localhost:3000/

To see the website.
